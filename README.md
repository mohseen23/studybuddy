📚 StudyBuddy

A peer-to-peer learning platform that helps students connect, collaborate, and support each other academically. StudyBuddy allows users to create profiles, search for study partners, and exchange knowledge without any financial cost.

🚀 Features

User registration and login
Create and manage academic profiles
Search users by subject
View study listings and details
Subject-based tags and categories
Dynamic content rendered from database

🛠️ Tech Stack

Frontend

HTML, CSS
PUG (templating engine)

Backend

Node.js
Express.js

Database

MySQL

DevOps

Docker
Git & GitHub
⚙️ Getting Started
Prerequisites

Make sure you have installed:

Docker
Docker Compose
Installation
