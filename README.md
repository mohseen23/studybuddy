StudyBuddy

StudyBuddy is a peer learning web application that helps students connect with each other for academic support. Users can create profiles, search for study partners, and offer or request help in different subjects.

Tech Stack
Node.js
Express.js
MySQL
PUG
Docker
Features
User registration and login
Create and view user profiles
Search users by subject
View study listings
Basic categories/tags for subjects
Setup

Clone the repository and run:

docker-compose up --build

Then open in browser:

http://localhost:3000
